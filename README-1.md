# بوت "العقل" — مساعد AI مع نظام ربح حقيقي عبر Telegram Stars

## 1) الفكرة باختصار
بوت يقدم مساعدة بالذكاء الاصطناعي (تلخيص، ترجمة، إجابة أسئلة، كتابة نصوص) بحصة يومية مجانية،
واشتراك شهري "بلا حدود" يُدفع بعملة تيليجرام الرسمية **Stars (XTR)** — بدون أي بوابة دفع بنكية،
مناسب 100% لمن هو في الجزائر أو أي دولة.

## 2) خطوات التشغيل

### أ. إنشاء البوت
1. افتح محادثة مع [@BotFather](https://t.me/BotFather) في تيليجرام.
2. أرسل `/newbot` واتبع التعليمات، ستحصل على `BOT_TOKEN`.
3. أرسل `/mybots` ← اختر بوتك ← Payments، تأكد أن الدفع بـ Stars مفعّل تلقائياً (لا يحتاج ربط بنك).

### ب. تجهيز البيئة
```bash
cd al3aql_bot
python -m venv venv
source venv/bin/activate   # على ويندوز: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
```
افتح `.env` وضع:
- `BOT_TOKEN`: التوكن من BotFather
- `ADMIN_IDS`: آيدي حسابك في تيليجرام (احصل عليه من بوت مثل @userinfobot)
- `AI_API_KEY`: مفتاح **Groq مجاني بلا بطاقة بنكية** — روح لـ [console.groq.com](https://console.groq.com)، سجل بإيميلك (Google/GitHub)، ثم من "API Keys" اعمل مفتاح جديد وانسخه. البوت مضبوط افتراضيا يخدم بـ Groq (`llama-3.3-70b-versatile`).

> إذا حبيت تبدل لخدمة ثانية لاحقا (OpenAI، DeepSeek...) بدّل فقط `AI_API_BASE` و`AI_MODEL` فـ `.env`، الكود متوافق مع أي API على نمط OpenAI بلا تعديل.

### ج. التشغيل
```bash
python bot.py
```
البوت يشتغل الآن. للتشغيل الدائم 24/7 استضفه على خادم رخيص (Contabo, Hostinger VPS) أو منصة مثل Railway/Render.

## 3) كيف تربح فعلياً (بدون بيع أي منتج)

| المصدر | كيفاش يشتغل |
|---|---|
| **Telegram Stars** | المستخدم يشتري اشتراك مميز بـ Stars داخل التطبيق مباشرة. أنت تحول الـ Stars لكاش حقيقي عبر [Fragment.com](https://fragment.com) (منصة تيليجرام الرسمية). |
| **إعلانات ممولة** | أصحاب قنوات/بوتات أخرى يدفعولك (عبر تحويل يدوي أو Stars) باش تعرض إعلانهم للمستخدمين المجانيين. استخدم `/addad نص الإعلان` كأدمن. |
| **نظام الإحالة** | ينمّي قاعدة المستخدمين مجانا (خصوصا إذا روّجت للبوت عبر TikTok/Instagram)، وكل مستخدم إضافي = فرصة اشتراك أو مشاهدة إعلان. |

## 4) أوامر البوت
- `/start` — بداية الاستخدام (يدعم رابط الإحالة `?start=ref_ID`)
- `/premium` — شراء الاشتراك المميز
- `/invite` — الحصول على رابط الإحالة الخاص بك
- `/stats` (أدمن فقط) — إحصائيات المستخدمين والأرباح
- `/addad <نص>` (أدمن فقط) — إضافة إعلان ممول

## 5) نصائح للنجاح
- روّج للبوت بفيديوهات قصيرة على TikTok توضح فائدة حقيقية (مثلا: "لخص واجباتك في ثانية").
- ابدأ بسعر منخفض للاشتراك (150 Star ≈ 3$) ثم ارفعه لما يكبر عدد المستخدمين.
- لا تكثر من الإعلانات الممولة حتى لا تُفقد المستخدمين المجانيين، ADS_EVERY_N_REPLIES في `config.py` يتحكم بالتكرار.
- التزم بقوانين تيليجرام: لا سبام، لا محتوى مخالف، الإعلانات يجب تكون واضحة أنها "ممولة".

## 6) ملاحظة مهمة
هذا الكود جاهز للتشغيل لكنه **إطار عمل احترافي** يحتاج منك:
- مفتاح AI API فعّال (بعض المزودين يوفرون رصيد مجاني بداية).
- استضافة تعمل 24/7 حتى يبقى البوت متاحاً للمستخدمين.
